Drafts and Demos

This repository is a place to store and work on interactive materials for Coded by Kids.

Page Build

A demo of an idea to walk kids through structuring HTML documents.

Baskets

A demo furthering the tag-testing idea, this time with link, image, and header tags.

Borders

A self-directed slideshow skeleton with placeholders.

Letter Drop

A simple hunt-and-peck typing game.

Color Changer

Page changes color with mouse movements, allowing rgb color selection.
